# Miss & Mister DefITech 2026

Plateforme de vote pour l'election de Miss & Mister DefITech 2026. Permet de parcourir les candidats avec photos et descriptions, voter via l'API JSONBin, et consulter le compte a rebours jusqu'a l'evenement.

## Stack technique

- HTML / CSS / JavaScript vanilla
- API JSONBin.io pour la persistance des votes
- Stockage local (fallback)

## Etat d'avancement

MVP -- fonctionnel pour l'evenement du 26 mars 2026.

## Demarrage

Ouvrir `index.html` dans un navigateur. Un serveur HTTP est recommande :

```bash
python -m http.server
# ou utiliser l'extension Live Server
```
